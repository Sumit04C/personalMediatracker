Project Setup:
1. Install dependencies from requirements.txt: pip install -r requirements.txt
2. Open the project folder using the command-line
3. Run app.py using the command 'python app.py'

You will then see a statement saying 'Running on http://127.0.0.1:5000' 

4. Once you click it, a new browser tab will open for you to add book entries. 

Note: This is a working first version of an ongoing project. 

Tools Used:
- Lucidchart
- Jira
- Flask, SQLite, and SQLAlchemy
